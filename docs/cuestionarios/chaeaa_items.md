# Cuestionario CHAEA de Estilos de Aprendizaje

Este archivo documenta las 80 afirmaciones utilizadas por el sistema SEEDU Motor Fine para el cuestionario **CHAEA** (Honey-Alonso). Cada ítem está asociado a una de las cuatro dimensiones: Activo (ACT), Reflexivo (REF), Teórico (TEO) o Pragmático (PRA), siguiendo la inicialización de `initialize-questionnaires`.

Formato de respuesta sugerido: **Sí / No** o **De acuerdo / En desacuerdo**.
